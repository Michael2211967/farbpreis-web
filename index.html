Farbpreisberechnung

10 INPUT "PREIS JE KILO FARBE EINGEBEN: ";KP Preis je Kilo Farbe eingeben:
20 INPUT "KILO JE QUADRATMETER EINGEBEN: ";KQ Kilo je Quadratmeter eingeben:
30 PRINT
40 INPUT "LAENGE EINGEBEN: ";L Länge eingeben:
50 INPUT "BREITE EINGEBEN:";B Breite eingeben:
60 F=L*B
70 M=F*KQ
80 P=M*KP
90 PRINT
100 PRINT "FUER DIE FLAECHE VON ";F;" QM"
110 PRINT "WERDEN ";M;" KILO FARBE BENOETIGT."
120 PRINT "DER PREIS FUER DIESE ";M;" KILO FARBE BETRAEGT ";P;" EUR.
130 END Ready.
